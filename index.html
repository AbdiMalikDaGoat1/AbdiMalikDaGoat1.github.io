My Screenprinting Project

An Intro to What This Project Is About?

I grew up in London, and for this project I wanted to make art that actually meant something to me. My name is Abdi, and this is my screen printing project about something I think a lot of people can relate to but never really talk about. The idea is simple. We are so caught up in our own lives that we stop noticing the world around us. This project is my way of saying slow down and look up.
